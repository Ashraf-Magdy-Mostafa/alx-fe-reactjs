# Recipe Sharing App (React + Zustand)

This project is structured to satisfy the ALX "State Management in React" tasks:
- Task 0: Initialize app + Zustand store + RecipeList + AddRecipeForm
- Task 1: Detailed recipe management (view details via route, edit, delete)
- Task 2: Advanced search & filtering
- Task 3: Favorites + personalized recommendations (mock)

## Run locally
1) Install deps:
   npm install

2) Start dev server:
   npm run dev

## Notes
- Components are under `src/components`
- Pages are under `src/pages`
- Store is `src/components/recipeStore.js`
