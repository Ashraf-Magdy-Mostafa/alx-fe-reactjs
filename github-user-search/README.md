# GitHub User Search (React + Vite)

## Setup
```bash
npm install
npm run dev
```

## Optional (Higher rate limits)
Create a `.env` file in the project root:
```bash
VITE_APP_GITHUB_API_KEY=YOUR_GITHUB_TOKEN
```
> Add the same variable to Vercel Environment Variables when deploying.

## Build
```bash
npm run build
npm run preview
```
